# Pixel Runner

Tiny endless-runner on a plain HTML `<canvas>`. Pure HTML/CSS/JS, no build-in tools.

## Play
- Open `index.html` in a browser.

## Controls
- ← / → : Move
- Space : Jump
- 2X Space : Double-Jump
- P : Pause
- R : Restart

Deployed via GitHub Pages: https://fenderbrender.github.io/pixel-runner/gamefile
